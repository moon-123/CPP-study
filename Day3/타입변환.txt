다른 타입끼리의 연산은 피연산자들을 모두 같은 타입으로 만든 후에 수행됨

자동 타입변환이 됨

    1. 다른 타입끼리의 대입, 산술 연산 시
    2. 함수에 인수를 전달할 때

범위가 좁은 타입에서 넓은 타입으로의 변환은 문제가 되지 않음
그 반대는 데이터 손실이 발생함

타입 변환의 종류
    1. 묵시적 타입 변환(자동 타입 변환)
    2. 명시적 타입 변환(강제 타입 변환)

묵시적 타입 변환(자동 타입 변환)
    대입 연산이나 산술 연산에서 자동으로 수행되는 타입 변환
    대입 연산 시 연산자의 (오른쪽에 존재하는 데이터의 타입)이 (왼쪽에 존재하는 데이터 타입)으로 (묵시적 타입 변환 진행)
    산술 연산 시에는 데이터의 손실이 최소화되는 방향으로 묵시적 타입 변환이 진행됨

char -> short -> int -> long -> float -> double -> long double


명시적 타입 변환(강제 타입 변환)
    (변환할타입) 변환할데이터 -> C와 C++ 모두 가능
    변환할타입 (변환할데이터) -> C++만 가능
