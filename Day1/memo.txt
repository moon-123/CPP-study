C++이란?
    - C언어에서 여러 가지 기능을 추가하여 만든 언어
    - C의 절차 지향적 특징을 가져옴
    - Simula에서 클래스를 사용하는 객체 지향적 특징을 가져옴
    - 일반화 프로그래밍 방식이라는 특징도 있음

증가 연산자(++)의 도입으로부터 C언어의 확장판이라는 의미를 담아 C++이라는 이름으로 개정함

C++ 특징 정리
    - 절차 지향적이며 구조적 프로그래밍 언어입니다.
    - 객체 지향 프로그래밍 언어입니다.
    - 일반화 프로그래밍 언어입니다.

C++ 프로그래밍 과정
    1. 소스 파일(source file)의 작성
        - .cpp 확장자의 파일 작성

    2. 선행처리기(preprocessor)에 의한 선행처리
        - 선행처리 문자(#)로 시작하는 지시문 처리 작업 

    3. 컴파일러(compiler)에 의한 컴파일
        - 컴파일, 컴파일이 끝나면 .o나 .obj

    4. 링커(linker)에 의한 링크
        - .o 파일들과 라이브러리 파일, 시동 코드를 하나로 합침
        - 분할 컴파일

    5. 실행 파일(executable file)의 생성
        - 소스 파일 -> 선행처리기 -> 컴파일러 -> 링커 -> 실행파일, .exe


C++ 표준 입출력 클래스
    - C++의 모든것은 객체로 표현됨
    - cout, cin

    - << 연산자는 연산자 오버로딩을 활용한 것임
