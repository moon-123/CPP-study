데이터를 저장하는 메모리 공간

숫자
    정수: char, int, long, long long
    실수: float, double

관련된 데이터를 한 번에 묶어서 처리하는 사용자 정의 구조체 변수도 있음

변수 생성 규칙
    - 영어, 숫자, _
    - 숫자로 시작 x
    - 공백 x
    - 키워드 x
    - 길이 제한 x

비트(bit)와 바이트(byte)
    - 컴퓨터는 모든 데이터를 2진수로 표현하고 처리함
    - bit는 최소단위
    - byte는 8개의 비트

변수와 메모리 주소
    - 변수는 메모리 주소를 기억하는 역할을 함
    - 변수를 참조할 떄는 메모리의 주소를 참조하는 것이 아닌, 해당 주소에 저장된 데이터를 참조하게 됨
    - 변수는 데이터가 저장된 메모리의 주소뿐만 아니라, 저장된 데이터의 길이와 형태에 관한 정보도 같이 기억해야함

변수의 선언
    - 변수를 저장하기 위한 메모리 공간을 할당받아야 함
    - 이를 변수의 선언이라고 부름
    
    변수 선언 방법
        - 변수의 선언만 하는 방법
            - 메모리 공간을 할당받고 나중에 초기화하는 방법
            - 해당 메모리 공간에는 알 수 없는 쓰레기값이 들어가 있음
            - 초기화하지 않은 변수는 사용하지 말자

        - 변수의 선언과 동시에 초기화하는 방법
